# KJEmitterView
---
* 这个工程提供了一些iOS开发中用到的类目，方便开发
* 这里有我经常用到的一些扩展，方便好用开发
* 整理一些好用的自定义控件，部分数据来源于网络 

---
### Cocoapods安装
```
pod 'KJEmitterView'
```
